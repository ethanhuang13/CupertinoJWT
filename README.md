## Install

### CocoaPods

You can use [CocoaPods](http://cocoapods.org/) to install `CupertinoJWT` by adding it to your `Podfile`:

```ruby
platform :ios, '10.0'
use_frameworks!

target 'MyApp' do
    pod 'CupertinoJWT'
end
```
